# Module: Platform LZ - Governance: Policy Definitions

This module processes a provided collection of JSON-formatted policy definitions files. 

Policy files names should be prefixed using the intended Management Group assignment name (e.g. `core_policy.json`, `platform_policy1.json`, `platform_policy2.json`). This ensures that policies containing the matching name prefix are added into the related Policy Definition Set.
